Bearcats Recruiting - Football Recruiting Platform

A full-stack recruiting platform that connects high school football players with college coaches. Players build detailed athletic profiles (height, weight, 40-yard dash, bench press, GPA, highlights) and coaches can browse, filter, and message prospects directly. Features real-time WebSocket messaging with unread badges, profile photo uploads, shareable profile links with iOS iMessage preview support via Open Graph meta tags, and a public player directory that requires no login to browse. Built with FastAPI, SQLite, and Jinja2 templates, served behind Nginx with SSL.
